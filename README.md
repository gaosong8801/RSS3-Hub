# RSS3 Hub

A centralized implementation of [RSS3](https://github.com/NaturalSelectionLabs/RSS3)

## API

### Persona

- GET `/persona/get/:id` - get a persona

- POST `/persona/add` - add a new persona

- POST `/persona/set`

- POST `/persona/del`

### Items

- GET `/items/get/:id`

- POST `/items/add`

- POST `/items/set`

- POST `/items/del`

## Link

- GET `/link/get/:id`

- POST `/link/add`

- POST `/link/del`
